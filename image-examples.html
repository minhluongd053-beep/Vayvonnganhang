Ví dụ chèn ảnh từ thư mục Pictures

Giả sử cấu trúc thư mục như sau (cùng cấp với file HTML):

  /Untitled-2.html
  /image-examples.html
  /Pictures/
    /Image/
      photo.jpg
    /Screenshots/
      shot.png
1) Chèn ảnh bằng thẻ <img> (đường dẫn tương đối):
<img src="Pictures/Image/photo.jpg" alt="...">
2) Chèn ảnh từ thư mục Screenshots:
<img src="Pictures/Screenshots/shot.png" alt="...">
3) Dùng ảnh làm background trong CSS:
Background image
background-image: url('Pictures/Image/photo.jpg');

Mẹo nhanh

  • Sử dụng dấu gạch chéo ngược (/) trong URL, luôn dùng forward slash trên web.
  • Nếu file HTML nằm trong cùng thư mục gốc như thư mục Pictures, đường dẫn bắt đầu bằng Pictures/....
  • Nếu đặt file HTML trong một thư mục con (ví dụ pages/index.html), phải điều chỉnh đường dẫn tương đối (ví dụ ../Pictures/Image/photo.jpg).
  • Tránh khoảng trắng trong tên file — nếu có khoảng trắng, mã hóa thành %20 hoặc đặt tên khác.
  • Khi thử nghiệm, nên chạy server (không chỉ mở file bằng file://) để tránh một số giới hạn CORS hoặc đường dẫn:
# Nếu có Python 3
python -m http.server 8000

# Hoặc trên Windows nếu dùng py
py -m http.server 8000

Sau khi chạy lệnh trên, mở trình duyệt vào http://localhost:8000 và duyệt tới image-examples.html.
